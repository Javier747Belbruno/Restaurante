List Stock Levels by Publisher Alternate
Details:
The List Stock Levels by Publisher allows a user to obtain stock level information for a selected publisher. The Stock Control Manager and Storeroom Worker need this information to plan logistics and to ensure that stock remains at adequate levels to service incoming requests. There is also the need to predict the date that the stock items will fall below an acceptable level based on purchase cycles and promotional periods.<br/>
Steps:
1. User selects "List Stock Levels by Publisher"<br/>
2. System returns a list of publishers to select from<br/>
3. User Selects a publisher<br/>
4. System returns a listing of titles and quantity in stock for the publisher<br/>
Update Stock levels Alternate
Details:
